# Theme Showcase - Quick Start

## Installation

The theme showcase requires the `recharts` library. Install it with:

```bash
cd starter-react-vite
pnpm add recharts
```

## Viewing the Themes

Start the development server:

```bash
pnpm dev
```

Then navigate to:

- **Light Only Theme**: [http://localhost:5173/theme-light](http://localhost:5173/theme-light)
- **Dark Only Theme**: [http://localhost:5173/theme-dark](http://localhost:5173/theme-dark)
- **Switchable Theme**: [http://localhost:5173/theme-switchable](http://localhost:5173/theme-switchable)

## Theme Comparison

### 1. Light Only (Professional Blue)
- Clean, professional look
- Deep blue primary (#0d47a1)
- Orange secondary (#f57c00)
- Best for business applications

### 2. Dark Only (Midnight Purple)
- Modern, sleek design  
- Vibrant purple primary (#7c4dff)
- Cyan secondary (#00e5ff)
- Best for creative/entertainment apps

### 3. Switchable (Current)
- Flexible light/dark toggle
- Blue primary in both modes
- User preference support
- Best for accessibility
